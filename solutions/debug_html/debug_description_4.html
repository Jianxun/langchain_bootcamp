Air Data, Altitude, Heading Reference Systems (ADAHRS)

Analog Devices provides all necessary components to complement precision data converters and provides a complete solution in the form of a µModule® using heterogeneous integration and system in package (SiP) technology. This enables the use of the best process for each stage in the signal chain, improving performance, reducing footprint, speeding time to market, and lowering the total cost of ownership.

Value and Benefits

Our precision signal chain µModules are data conversion solutions based on heterogeneous integration. Debugging or optimizing external circuitry is not required.

Safely operates commercial and military aviation with AHAHRS measurements

Provides high-performance IMUs, gyros and accelerometers with inertial MEMS technology

Enables high accuracy, low noise air data sensing with precision data acquisition product
